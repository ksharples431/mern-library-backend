My Books API: Documentation

About this API

A simple API that provides users with information about books, authors, and genres.

The Api's features include

  • Returns list of all books
  • Returns data about any single book including description, author, genre
  • Returns data about a specific author
  • Returns data about a specific genre
  • Allows new users to register
  • Allows users to update thier user info
  • Allows users to add a book to a list of their favorites
  • Allows users to remove a book from their favorites list
  • Allows existing users to deregister

Endpoints

Business Logic URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Returns a list of all books /books GET - - A JSON object holding data about all the books
Returns data about a single book by title /books/:Title GET :Title - A JSON object holding data about a single book
Returns data about an author by author name /books/authors/:authorName GET :authorName - A JSON object holding data about a specific author
Returns data about a genre by genre name /books/genres/:genreName GET :genreName - A JSON object holding data about a specific genre of book
Allows new users to register /users POST - A JSON object holding data about the user to add, structured like: { Username: req.body.Username, Password: req.body.Password, Email: req.body.Email, Birthday: req.body.Birthday } A JSON object holding data about the user that was added
Allows users to update their user info /users/:Username PUT :Username - A JSON object holding data about the updated user information
Allows users to add a book to their favorites /users/:Username/books/:BookID POST :Username, :BookID - A JSON object holding data about the user and the book that was added
Allows users to remove a book from their favorites /users/:Username/books/:BookID DELETE :Username, :BookID - A JSON object holding data about the user and the book that was removed
Allows existing users to deregister /users/:Username DELETE :Username - A text message indicating whether the user has successfully deregistered
